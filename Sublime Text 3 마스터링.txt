2. 에디터 기본 설정 및 단축키 변경  
	- 파일 생성 및 사이드바 설정   
		새 문서 파일을 생성하고 저장하는 방법을 다루고, 파일 확장자에 따른 문법 모드 변경 및 직접 문법 모드를 변경하는 방법을 다뤄봅니다. 사이드 바를 토글하는 기본 단축키 설정을 확인하고, 변경하는 방법에 대해 이야기합니다.  

		진행내용 
		•새 문서(New File) 생성/저장
		•문법(Syntax) 모드 변경
		•사이드 바(Side Bar) 토글(Toggle, 보이기/감추기)
		•사이드 바에 폴더(Directory)를 드래그(Drag)하여 관리

	- 단축키 확인 변경 & 에디터 설정 변경   
		에디터 단축키 기본값을 확인하거나, 사용자 정의 덮어쓰기하는 방법을 다뤄봅니다. 그 외 다양한 설정을 통해 에디터를 사용자화 하는 방법을 상세히 공부해봅니다. 

		진행내용 
		•단축키 기본값 확인/덮어쓰기 - Key Bindings (Default & User)
		•사이드 바 토글(Toggle Side Bar) 단축키 변경
		•설정 기본값 확인/덮어쓰기 - Setting (Default & User)
		•타이포그래피 설정 변경 - font_size, font_face, font_options
		•패딩 설정 변경 - line_padding_top, line_padding_bottom
		•탭 사이즈 설정 변경 - tab_size
		•워드 랩 설정 변경 - word_wrap
		•미니 맵 설정 변경 - draw_minimap_border, always_show_minimap_viewport
		•하이라이트 라인 설정 변경 - hightlight_line
		•스크롤 설정 변경 - scroll_past_end

4. '패키지 관리 도구' & '사이드바 강화' 패키지 설치/활용  
	- Package Control 및 Sidebar Enhancements 패키지 설치  
		패키지 관리를 용이하게 도와주는 유용한 도구 Package Control을 설치하고, Install Package 명령어를 통해 Package를 검색/설치하는 방법을 다룹니다. 뿐만 아니라 첫번째 설치할 Package로 사이드바를 보다 강력하게 사용하도록 도와주는 Sidebar Enhancement 패키지를 설치해봅니다. 

		진행내용 
		•패키지 관리 도구(Package Control) 설치
		•인스톨 패키지(Install Package) 명령어로 패키지 검색/설치
		•사이드바 강화(Sidebar Enhancement) 패키지 설치

		[Package Control 설치]
		https://sublime.wbond.net/ 

		- Sidebar Enhancements 사용자 정의 단축키 설정  
		사이드바 사용시, 자주 사용되는 기능을 단축키로 등록 활용하는 방법을 다뤄봅니다. 

		진행내용 
		•[F12]를 눌러 열 기본 브라우저 설정
		•[Ctrl+Alt+N]을 눌러 새 파일 생성
		•[F2]를 눌러 파일 이름 변경
		•[Ctrl+Alt+M]을 눌러 파일 위치 이동
		•[Ctrl+Alt+D]를 눌러 파일 제거

		[구글 크롬 웹 브라우저 다운로드]

11. 자동완성 - 코드힌트/스니펫/매크로 활용  
	- 내장된 자동 완성(Auto Complete) 활용    
		Sublime Text가 내장하고 있는 자동 완성 기능에 대해 설명합니다. 

	- 스니펫(Snippet) 활용    
		Sublime Text의 강력한 스니펫 기능을 활용한다면 여러분 업무 프로세스가 향상될 것이라고 자신합니다! 

	- 매크로(Macro) 활용    
		일련의 행동을 반복 수행하는 노가다를 줄이고 싶다면? Sublime Text의 매크로 기능을 공부해보세요! 당신의 고민을 말끔히 해결해줄겁니다! :-) 

	- 매크로(Macro) 활용 - EditPlus3 '글자 복제' 기능 만들기    
		Sublime Text 3에는 현재 EditPlus 3의 '글자 복제' 같은 기능을 제공하지는 않습니다. 하지만 매크로(Macro)를 사용하면 동일한 기능을 추가할 수 있을 뿐 아니라, 단축키 또한 사용자 정의하여 설정할 수 있습니다. 

		혹시 바로 사용해보시고 싶으시면 제가 만든 매크로 파일을 다운로드 받아 테스트 해보세요. 다운 받은 Macro.zip 파일의 압축을 푼 후, duplicate-char(like EditPlus3).sublime-macro 파일을 포함한 Macro 폴더를 Package/User 안에 복사 붙여넣기하면 사용할 수 있습니다. 

		그리고 단축키 설정은 'Preference > Key Bindings - User'에 아래와 같은 코드를 입력해주면 사용 가능합니다. 

		// Macro 
		{ "keys": [ "ctrl+-" ], "command": "run_macro_file", "args": {"file":"Packages/User/Macro/duplicate-char(like EditPlus3).sublime-macro"} },

	- OS '컬러 선택도구'를 팝업으로 띄우는 Color Picker      
		쉬운 설치 방법
		영상 강의가 어려우신 분들은 아래 ZIP 파일을 다운로드 받아 압축을 푼 후, ClipboardHistory.sublime-package 파일을 아래 경로에 붙여넣기 합니다. (Sublime Text 3 64bit 기준) 

		'C:\Users\사용자계정\AppData\Roaming\Sublime Text 3\Installed Packages' 

		Sublime Text는 기본적으로 에디터 상에서 컬러 선택도구(Color Picker)를 제공하지 않아 불편합니다. 이 문제를 개선시키기 위해서 Color Picker 패키지를 설치하면 OS에서 제공하는 '컬러 선택도구'를 창으로 띄울 수 있습니다. 

		[Color Picker 패키지]
		github.com/weslly/ColorPicker

13. 빠른 HTML마크업/CSS스타일링을 위한 Package! - Emmet  
	- Emmet Package 설치    
		HTML 마크업, CSS 스탕일링을 빠르게 처리가능하도록 도와주는 강력한 Emmet 패키지를 설치하는 과정을 알아봅니다. 

		[Emmet.io 웹사이트]
		emmet.io 

	- Emmet HTML - '표준 DTD 문서' 유형별 생성    
		HTML 표준 문서인 HTML4.01, XHTML1.0 그리고 새로운 표준으로 자리잡고 있는 HTML5 문서 기본형을 빠르게 작성하는 방법을 알아봅니다.  

		[Emmet Documentation]
		docs.emmet.io  

		[HTML 문서 기본형 축약코드]
		doc → HTML5 문서 기본 골격 [DTD 제외] 
		doc4 → HTML4 문서 기본 골격 [DTD 제외]
		html:4t → HTML4 문서 관용모드 기본 골격
		html:4s → HTML4 문서 엄격모드 기본 골격
		html:xt → XHTML1 문서 관용모드 기본 골격
		html:xs → XHTML1 문서 엄격모드 기본 골격
		html:5 또는 ! → HTML5 문서 기본 골격  

		[DTD 축약코드]
		!!!4t → HTML4 관용모드 DTD 
		!!!4s → HTML4 엄격모드 DTD 
		!!!xt → XHTML1 관용모드 DTD 
		!!!xs → XHTML1 엄격모드 DTD 
		!!! → HTML5 DTD 
 
	- Emmet HTML - '요소 & 형제/자식요소' 생성    
		[연산자 + > ]
		HTML 요소 및 형제, 자식 요소를 빠르게 생성하는 방법을 익혀봅니다. 

		[요소/형제/자식요소 만드는 축약코드 ]
		요소이름 [tab] 
		형제 선택자 + 
		자식 선택자 > 

	- Emmet HTML - 구문 연결을 위한 '부모요소 접근 및 그룹핑'    
		[연산자 ( ) ^] 
		코드 생성구문을 이어서 작성하기 위해 부모요소에 접근하는 방법과 그룹핑하는 방법을 익혀봅니다.  

		[그룹핑/부모요소 접근 연산자 ]
		그룹핑 - () 
		부모요소 접근 - ^ 

	- Emmet HTML - 복수 요소를 생성하기 위한 연산자    
		[연산자 *] 
		요소를 1개 이상 복수를 만들고자 할 때 사용 가능한 Emmet 연산자 활용법을 익혀봅니다. :-) 
 
	- Emmet HTML - ID, Class, 속성 선택자를 통한 생성법    
		[연산자 # . [ ] ]
		CSS 속성 선택자를 활용한 요소 생성 및 단축 속성 ID, Class를 활용하여 빠르게 HTML 마크업 하는 방법을 알아봅니다.

	- Emmet HTML - 숫자를 자동으로 생성시키는 연산자    
		[연산자 $ @ - ]
		$, @ 연산자를 활용하여 자동으로 숫자를 생성하고 순차적으로 또는 역순 방향으로 나열하는 방법을 알아봅니다. 뿐만아니라 생성할 숫자의 시작 값을 설정하는 방법 역시 다뤄봅니다. 

	- Emmet HTML - 텍스트를 추가하는 연산자    
		[연산자 { } ]
		Emmet 코드 구문을 작성하면서 요소만 생성하는 것이 아니라, 각 요소 내부에 텍스트를 생성하는 방법을 알아봅니다. 
 
	- Emmet HTML - 부모요소에 따라 암묵적 코드 생성    
		[Implicit tag names]
		명시하지 않아도 암묵적으로 부모 요소의 유형에 따라 자동으로 태그가 결정되는 기능을 Emmet은 지원합니다. 
 
	- Emmet HTML - 더미텍스트(Lorem Ipsum) 생성기 활용    
		["Lorem Ipsum" Generator]
		디자인 프로토타입 제작 시, 더미 텍스트를 생성해주는 기능을 Emmet은 지원합니다. 
 
	- Emmet HTML - 커스터마이징 (맞춤 설정)    
		[Emmet Customizing]
		Snippets.json 파일에 설정되어 있는 Snippets, Abbreviations, Alias 를 사용자 정의하에 설정하는 방법을 공부해봅니다. 또한 문법 출력모드 변경에 대해서도 알아봅니다. 
 
	- Emmet HTML - 주언어 설정 값을 한국어(ko-KR)로 변경 설정    
		HTML 문서의 '주언어' 설정을 영어(en)에서 한국어(ko)로 기본 설정 값을 바꿔봅니다. 
 
	- Emmet HTML - 인터랙티브한 코드 작성법, 축약코드로 감싸는 방법, 태그 제거 방법    
		Emmet으로 작성 가능한 빠른 HTML 마크업의 진수이자, 결정판입니다. 아래 코드를 필히 마스터해보세요! 

		[인터랙티브 코드 작성: Interactive Expand Abbreviation]
		Ctrl + Alt + Enter  

		[축약 코드로 감싸서 코드 작성: Wrap with Abbreviation]
		Ctrl + Shift + G  

		[태그 제거: Remove Tag]
		Ctrl + Shfit + ; 

	- Emmet CSS - 속성, 값, 단위, 컬러 등을 축약 코드로 생성    
		Emmet에서 사용 가능한 CSS 축약 코드 생성 방법을 익혀봅니다. 
		문법, 단위, 컬러, !important 등 축약 코드 활용 
 
	- Emmet CSS - 브라우저 제조사 접두사 코드 자동 생성    
		새로운 CSS3 속성마다 브라우저 제조사의 접두사를 붙이는 것은 매우 곤혹스러운 일이죠. 이 문제를 Emmet은 멋지게 해결해줍니다. :-) 그 방법을 익혀보겠습니다. 
 
	- Emmet CSS - CSS3 Gradients 코드 자동 생성    
		CSS3 Linear-Gradient 문법을 잘 모르더라도 Emmet을 활용하면 손쉽게 크로스 브라우징 그레디언트 코드를 생성할 수 있습니다. 
 
	- Emmet CSS - 빠른 코딩(Fuzzy Search) & 커닝페이퍼(Cheat Sheet)    
		Emmet을 활용하면 CSS 스타일링도 매우 빠르게 작성이 가능합니다. 물론 익숙하지 않은 방법을 익히기 위해서는 학습이 필요하기 때문에 커닝페이퍼의 도움을 받아 여러번 반복해주는 것이 중요합니다. 

		[Emmet Cheat Sheet]
		docs.emmet.io/cheat-sheet/
